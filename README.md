# Sorteio e Soma dos Valores Pares

Este projeto gera cinco números aleatórios, armazena-os em uma lista e calcula a soma de todos os valores pares.

## Funcionalidades

- Sorteia 5 números aleatórios.
- Armazena os números em uma lista.
- Exibe os números sorteados.
- Calcula e exibe a soma dos valores pares.

## Tecnologias

- Python 3

## Conceitos praticados

- Funções
- Listas
- Estruturas de repetição
- Condicionais
- Módulo `random`
- Módulo `time`
